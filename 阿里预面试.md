# linux

## 符号的用法

### 1、& (表示在后台运行)

```bash
nohup java -jar demo.jar > test.log &
```

### 2、&& (表示前一条指令执行成功，才执行后一条)

```bash
cd test && cd ..
```

### 3、| (管道，上一条命令的输出作为下一条命令的输入)

```bash
netstat -nlp | grep xxx
```

### 4、|| (表示前一条命令失败后，才执行下一条命令)

```bash
cd test || cd mydata
```
